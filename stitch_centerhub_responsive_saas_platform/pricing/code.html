Can we migrate existing student data?

Absolutely. We provide automated CSV and JSON import tools. For Enterprise clients, our white-glove migration team handles the entire transition to ensure zero downtime.

How does Multi-Tenancy work in CenterHub?

Our Enterprise architecture allows you to manage multiple centers or franchises from a single master dashboard while maintaining strict data isolation between individual branches.
